# HR Analytics Dashboard

## Project Overview
This HR Analytics Dashboard was built using Power BI to analyze workforce trends, employee attrition, demographics, and hiring insights.

The dashboard helps HR teams understand employee distribution, attrition patterns, gender diversity, department performance, and workforce demographics.

---

## Tools Used
- Power BI
- DAX
- Power Query
- CSV Dataset

---

## Dashboard Pages

### 1. Executive Overview
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Department Workforce Analysis
- Hiring Trend
- Gender Distribution

### 2. Attrition Analysis
- Attrition by Department
- Attrition by Age Group
- Attrition by Gender
- Attrition Trend

### 3. Workforce Demographics
- Gender Breakdown
- Race Distribution
- Job Title Analysis
- Average Age Analysis

---

## Key Insights
- Engineering department has the highest workforce.
- Attrition is highest among employees aged 26–35.
- Male employees slightly dominate the workforce.
- Workforce diversity analysis highlights racial distribution across departments.
